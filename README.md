# LifeTrack

个人成长追踪 Web 应用 —— 目标管理、任务追踪、课程表、习惯打卡、睡眠记录、专注计时、心情日记一站式平台。

## 核心功能

| 模块 | 功能 |
|------|------|
| 📊 总览 Dashboard | 快捷操作、心情记录、习惯打卡、课程完成、目标进度、智能推荐、通知设置 |
| 📝 任务 Task | 优先级管理、批量操作、目标关联、截止日期、搜索筛选、优先级颜色条 |
| 📅 课程表 Schedule | 周/月双视图、拖拽安排、时间冲突检测、当前时间线、Pomodoro 专注 |
| 🎯 目标 Goal | 进度追踪、过期提醒、课程时间投入统计 |
| 😴 睡眠 Sleep | 时长趋势、质量评分 |
| ✅ 习惯 Habit | 热力图打卡、连续天数统计 |
| 📈 数据 Stats | 专注趋势、工作量统计、睡眠-专注关联分析、成就徽章 |
| 🤖 AI 助手 | 本地 Ollama 集成，数据不出设备 |
| ⚙️ 设置 Settings | 主题切换、导航自定义、JSON/CSV/Excel 导入导出、数据迁移 |

## 创新亮点

1. **智能学习路径推荐** —— 基于目标完成度、习惯 streak、专注记录等数据，自动生成个性化学习建议
2. **睡眠-专注关联分析** —— 皮尔逊相关系数分析睡眠质量与次日专注力的数据关联，找出最佳作息区间

React 19 + TypeScript + Vite + Tailwind CSS + Dexie (IndexedDB)

## 快速开始

```bash
cd lifetrack-web
npm install
npm run dev
```

浏览器访问 `http://localhost:5173/`

---

## 核心交互

| 操作 | 方式 |
|------|------|
| 切换页面 | 底部导航栏 或 **Ctrl+1~8** 快捷键 |
| 全局搜索 | **Ctrl+K** 快速定位 |
| 把任务排进课程表 | 课程表页 → 展开「任务」面板 → **拖拽** 任务到日程格子 |
| 调整课程时间 | **拖拽** 课程卡片到新的日期/时间段 |
| 标记任务完成 | 任务页点击状态圆环 / Dashboard 快捷操作 |
| 批量操作 | 任务页点击「批量」→ 多选 → 标记完成 / 分配目标 / 删除 |
| 数据备份 | 设置页 → 导出 JSON / 导出 CSV |
| 离线使用 | PWA 添加到主屏幕后，无网络也能正常使用 |
